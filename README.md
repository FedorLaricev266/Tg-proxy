# TG proxy by Jr.Larzyk

**TG proxy by Jr.Larzyk** — это open-source Android-приложение для локального Telegram proxy.

Приложение запускает локальный прокси на Android-устройстве и позволяет подключить Telegram одной кнопкой.

## Возможности

* Запуск прокси одной кнопкой
* Простой Android-интерфейс
* Уникальный Secret для каждого устройства
* Автоматическое открытие Telegram
* Поддержка локального proxy-режима
* Поддержка Cloudflare-домена
* Настройки таймаутов
* Режим пониженного энергопотребления
* Оптимизации для скорости и низкой задержки
* Открытый исходный код

## Как это работает

Приложение запускает локальный прокси-сервер на устройстве.

Telegram подключается к локальному адресу:

```text
127.0.0.1:<PORT>
```

После этого приложение перенаправляет трафик через настроенный proxy-маршрут.

При первом запуске приложение генерирует собственный Secret для устройства. Он сохраняется локально и используется повторно, пока пользователь не создаст новый Secret в настройках.

## Требования

Для сборки проекта нужны:

* Android Studio
* Android SDK
* JDK 17 или новее
* Gradle Wrapper из проекта

Обычно отдельная установка Gradle не требуется, потому что проект содержит `gradlew` и `gradlew.bat`.

## Сборка APK

Открой терминал в папке проекта.

### Windows PowerShell

```powershell
cd C:\Projects\tgws_android_app
$env:JAVA_HOME = "C:\Program Files\Android\Android Studio\jbr"
$env:Path = "$env:JAVA_HOME\bin;$env:Path"
.\gradlew.bat assembleDebug
```

### Linux / macOS

```bash
cd ~/Projects/tgws_android_app
export JAVA_HOME="/path/to/jdk"
export PATH="$JAVA_HOME/bin:$PATH"
chmod +x ./gradlew
./gradlew assembleDebug
```

После успешной сборки APK будет здесь:

```text
app/build/outputs/apk/debug/app-debug.apk
```

На Windows этот же путь выглядит так:

```text
app\build\outputs\apk\debug\app-debug.apk
```

## Release-сборка

Для создания release APK:

### Windows

```powershell
.\gradlew.bat assembleRelease
```

### Linux / macOS

```bash
./gradlew assembleRelease
```

Неподписанный release APK будет здесь:

```text
app/build/outputs/apk/release/app-release-unsigned.apk
```

Для публичного распространения release APK нужно подписать своим keystore.

## Установка APK через ADB

Подключи Android-устройство по USB и включи отладку по USB.

Проверь подключение:

```bash
adb devices
```

Установи APK:

```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

Для Windows можно использовать такой путь:

```powershell
adb install -r app\build\outputs\apk\debug\app-debug.apk
```

## Настройки приложения

В настройках доступны:

* локальный порт;
* Cloudflare-домен;
* таймаут подключения;
* idle timeout;
* копирование Telegram-ссылки;

## Как пользоваться

1. Установи APK.
2. Открой приложение.
3. Нажми большую кнопку Telegram.
4. Приложение запустит прокси.
5. Telegram откроется с готовой proxy-ссылкой.
6. Подтверди подключение прокси в Telegram.

## Энергопотребление

Расход батареи зависит от качества сети и активности Telegram.

Примерные значения:

```text
Прокси включён, Telegram почти не используется: 1–3% в сутки
Обычное использование Telegram: 2–7% в сутки
Активные чаты и медиа: 7–15% в сутки
Постоянные загрузки или видео: 18%+ в сутки
```

Для лучшей стабильности можно отключить ограничения батареи для приложения в настройках Android.

## Приватность
